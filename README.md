The markdown code provided estimates crossing rates for a variety of tagged fishes within an acoustic telemetry array. 
The five species analyzed here are: shorthead redhorse, white bass, channel catfish, freshwater drum, and smallmouth buffalo.
The acoustic array was located in the Sanduskry River (OH), Lake Erie and other Lake Erie Tributaries. 
Data for this code is provided by USGS - Lake Erie Biological Station, icluding the detection data (LENAT_detectionsWithLocs_20231128_200102.csv) and positional data (positions2023_all.csv).
Shapefiles used for the crossing analysis are provided here as, barrier buffer.shp.
